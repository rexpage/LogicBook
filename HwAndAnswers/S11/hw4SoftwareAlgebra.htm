  5. Define doublecheck properties that check to see whether the concatentation operation (append) preserves elements in lists supplied as operands. That is, any element that occurs in either of the lists xs and ys must also occur in the list (append xs ys), and vice versa. Formulate your definitions as implications. ACL2 includes all standard logical operators as instrinsics. For example, "and", "or", "not", and "implies" are instrinsics in ACL2 and have the meanings you would expect from classical logic. Another intrinsic that you will find useful in this exercise is "member-equal". The formula (member-equal x xs) is true (that is, not nil) if the value x occurs in the list xs. Otherwise, it is false (that is, nil).
